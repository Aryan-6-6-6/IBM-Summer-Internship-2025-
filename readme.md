# 🌊 Marine Life AI Assistant

A specialized AI assistant designed to provide expert information on marine biology, underwater life, and aquarium care.

## Features
- **Context-Aware:** Upload PDFs or DOCX files for specific analysis.
- **Specialized Persona:** Maintains a warm, non-robotic tone focused strictly on marine life.
- **Streaming UI:** Real-time response generation using Gradio.
